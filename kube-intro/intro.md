# Kubernetes History

## Why was Kubernetes created? What problem does it solve that Docker alone cannot?
Docker is basically used to containerize application and run the application in any environment. It solves "works on my machine" problem. Kubernetes is container orchestration
tool that enables self-healing, scaling, rolling updates and rollbacks of containers.

## Who created Kubernetes? What was it inspired by?
Google created kubernetes and open-sourced it in 2014. Later it was donated to Cloud Native Computing Foundation (CNCF), where it is now maintained. 

Kubernetes was heavily inspired by google's internal cluster management system called Borg, used to manage containers. Google engineers took the concept from it, simplified
them and made them open-source. That became kubernetes.

## What does kubernetes mean?
The word kubernetes comes from Greek "kybernetes", which means ship pilot. Kubernetes is literally the pilot of your containers.
